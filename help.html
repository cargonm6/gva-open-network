🚀 Empezar a usar ONetMap

ONetMap es un editor visual donde puedes crear redes de dispositivos arrastrando, conectando y organizando elementos directamente en el lienzo.

Todo ocurre en el canvas central. La barra izquierda contiene herramientas y la derecha muestra propiedades del elemento seleccionado.

🖥️ Añadir dispositivos a la red

Para añadir un dispositivo:

  1. Selecciona una herramienta (Router, Switch, PC o Patch Panel)
  2. Haz click en el lienzo
  3. El dispositivo aparecerá automáticamente
💡 Tip

Puedes colocar varios dispositivos rápidamente sin cambiar de herramienta.

🔗 Conectar dispositivos

Para crear una conexión:

  1. Selecciona la herramienta Enlace
  2. Haz click en el primer dispositivo
  3. Haz click en el segundo dispositivo

El sistema dibuja automáticamente la conexión entre ambos nodos.

⚠️ Importante

No puedes crear enlaces sin seleccionar dos dispositivos diferentes.

📦 Organizar tu red con áreas

Las áreas te permiten agrupar dispositivos visualmente.

  1. Selecciona la herramienta Área
  2. Haz click en el lienzo
  3. Ajusta el tamaño arrastrando la esquina inferior derecha
💡 Uso recomendado

Usa áreas para separar salas, edificios o zonas de red.

📝 Añadir notas o texto

Puedes añadir notas directamente en el diagrama.

  1. Selecciona la herramienta Texto
  2. Haz click en el lienzo
  3. Escribe el contenido
✏️ Editar texto

Haz doble click sobre el texto para editarlo nuevamente.

🖱️ Seleccionar y mover elementos

Selecciona cualquier elemento haciendo click sobre él.

  • Arrastra para mover dispositivos
  • Haz click en un área para seleccionarla
🔴 Indicador visual

El elemento seleccionado se resalta en rojo.

⚙️ Editar propiedades

Cuando seleccionas un elemento, verás sus propiedades en el panel derecho.

  • Cambiar ID
  • Cambiar nombre
  • Editar texto (si aplica)
⚠️ Validaciones

No puedes usar IDs duplicados dentro del proyecto.

🔍 Navegar en el lienzo

  • Usa la rueda del mouse para hacer zoom
  • Usa el slider izquierdo para ajustar zoom
  • Botón central del mouse para mover el lienzo
🎯 Consejo

El zoom se centra en la posición del cursor.

💾 Guardar y cargar proyectos

📤 Exportar

  • JSON → editable
  • GZIP → comprimido
  • PNG → imagen del diagrama

📥 Importar

Puedes cargar archivos .json, .gz o .gzip.

💡 Consejo

Guarda frecuentemente si estás trabajando en diagramas grandes.

⌨️ Atajos rápidos

  • Ctrl + C Clonar dispositivo
  • Enter Guardar texto
  • Esc Cancelar herramienta
  • Delete Eliminar elemento
⚡ Flujo rápido

Puedes usar Ctrl + C para duplicar dispositivos seleccionados rápidamente.
